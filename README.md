# notes

A [digital garden](https://notes.andrestorhaug.com/) — a collection of interconnected notes published as a supplement to [my personal website](https://www.andrestorhaug.com/).

## What is this?

This is a public subset of notes maintained in a private [Obsidian](https://obsidian.md/) vault. Rather than polished blog posts, these are living documents that grow and evolve over time.

## Structure

```
notes/       — published notes (flat, linked with [[wikilinks]])
templates/   — Obsidian templates for new notes
```
